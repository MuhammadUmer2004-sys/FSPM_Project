### **Level 3: Work Packages (Detailed Tasks)**
These are the actual "doable" items you and your team of 5 worked on.

#### **1.1 Project Initiation & Planning**
*   1.1.1 Develop Project Charter & SMART Objectives
*   1.1.2 Define Scope Boundaries (In-Scope vs. Out-of-Scope)
*   1.1.3 Create Initial Project Schedule (AoN Network Diagram)
*   1.1.4 Conduct Cost Estimation (Final Budgeting & Contingency)

#### **1.2 Design & Architecture**
*   1.2.1 UI/UX Mockups (Dark Mode & Glassmorphism Design)
*   1.2.2 System Architecture Design (React + Express + MongoDB)
*   1.2.3 Data Schema Mapping (Task & Bid JSON structure)

#### **1.3 core Development (FSPM Features)**
*   1.3.1 **Management Module**: W5HH Input Form & Scope Boundary UI
*   1.3.2 **Estimation Module**: COCOMO-lite suggestion tool
*   1.3.3 **Bidding Module**: Milestone-based phased bidding (WBS mapping)
*   1.3.4 **Auth Module**: Role-Based Access Control (RBAC) – Client vs. Developer
*   1.3.5 **Process Module**: Audit Trail logging & Database Integration

#### **1.4 Quality Assurance & Testing**
*   1.4.1 Unit Testing for Core API Endpoints
*   1.4.2 Browser "Smoke Testing" for core flows (Post -> Bid -> View)
*   1.4.3 Risk Audit & Review of Risk Register mitigation plans

#### **1.5 Project Closure & Presentation**
*   1.5.1 Final Portfolio Compilation (PMP Document)
*   1.5.2 Project Deck/Presentation Slides construction
*   1.5.3 Final Code Deployment & Demo readiness
